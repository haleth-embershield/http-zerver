About HTTP-Zerver

This is a minimal HTTP server written in Zig without using the standard library, designed specifically for Windows.

It serves static files from a local directory and demonstrates low-level networking and file I/O in Zig.

New Features

Directory Listing

The server now supports directory listing! When you request a directory without an index.html file, the server will generate a nice HTML page showing all files and subdirectories.

Try it out by:

  1. Creating a directory without an index.html file
  2. Navigating to that directory in your browser

Verbose Logging

You can now enable verbose logging to see detailed information about requests and responses.

Start the server with the -v or -verbose flag:

zig build run -- 8000 . -v

This will show:

  • Detailed request information
  • File paths being accessed
  • Response headers
  • Connection status

Command Line Usage

The server can be started with the following options:

zig build run -- [PORT] [DIRECTORY] [-v|-verbose]

Where:

  • PORT: The port to listen on (default: 8000)
  • DIRECTORY: The directory to serve files from (default: current directory)
  • -v|-verbose: Enable verbose logging

Example

To serve files from the "public" directory on port 9000 with verbose logging:

zig build run -- 9000 public -v
